Pocketsphinx.js test suite with Chinese acoustic model

This test suite uses one acoustic model for Chinese packaged with PocketSphinx on the subversion directory. The JavaScript file can be compiled with:

 $ cmake .. -DEMSCRIPTEN=1 \
   -DCMAKE_TOOLCHAIN_FILE=path_to_emscripten/cmake/Platform/Emscripten.cmake \
   -DHMM_BASE=path_to_pocketsphinx/model/hmm/zh \
   -DHMM_FOLDERS=tdt_sc_8k -DALLOW_MEMORY_GROWTH=1

It uses the file ../webapp/js/pocketsphinx_zh.js.

A pronunciation dictionary for Chinese based on this phone set can be found in PocketSphinx on the subversion directory in model/lm/zh_CN/mandarin_notone.dic.
